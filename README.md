# scorpio-analyst

Your personal portfolio manager and financial analyst

Inspired by the paper [arXiv:2412.20138](https://arxiv.org/pdf/2412.20138), this project is a Rust-native
reimplementation of the TradingAgents framework (originally
Python/LangGraph, [TauricResearch/TradingAgents](https://github.com/TauricResearch/TradingAgents)). It
simulates a trading firm with specialized agent roles, orchestrated through a multi-phase execution pipeline.

Please check [PRD.md](PRD.md) for more details on the project specification and architecture. The system is in early
development, so expect frequent updates and changes. Contributions are welcome!
